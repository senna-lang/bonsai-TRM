## Context

Phase 0-3 の実験が完了し、ベースラインとスキャフォールドの比較結果が揃った。Phase 3 の Decision Point 3 で記事の方向性が決定されている。

## Goals / Non-Goals

**Goals:**
- Phase 0-3 の結果を Zenn 記事として執筆
- 論文との比較は参考値として誠実に扱う(自環境での再現なし)
- GitHub リポジトリを公開可能な状態に整備
- 結果のシナリオに応じた適切なフレーミング

**Non-Goals:**
- 英語版記事の同時執筆(オプション)
- 論文の完全な再現報告(Bonsai での実験が主軸)
- 追加実験の実施

## Decisions

### 1. 記事の主軸は Bonsai 内 Before/After
**選択**: ベースライン vs スキャフォールドの比較を主軸にする
**理由**: Qwen3-8B AWQ を自環境で再現していないため、論文数値との直接比較は参考扱い。

### 2. 結果がどのシナリオでも記事として成立させる
**選択**: 楽観(「1bit でも効く」)、標準(「効果はあるが規模が小さい」)、悲観(「限界の分析」)のいずれでも書ける構成
**理由**: 実験プロジェクトでは結果を予測できないため、事前に構成を柔軟に設計。

### 3. Qwen3-8B との比較は参考値として明記
**選択**: 論文の数値は引用するが、環境差を含む可能性を明記
**理由**: 誠実な科学コミュニケーション。

## Risks / Trade-offs

- **[結果が面白くない]** → 0% → 0% の場合でも「限界の分析」として記事化。失敗モードの詳細分析で価値を出す
- **[記事が長くなりすぎる]** → 対策: 技術詳細は GitHub README に逃がし、記事は核心に集中
