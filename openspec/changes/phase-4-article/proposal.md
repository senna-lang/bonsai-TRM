## Why

実験結果を Zenn 記事として公開し、1-bit LLM での推論時スキャフォールドの有効性(または限界)を共有する。結果のシナリオに関わらず記事として成立する設計。GitHub リポジトリも整備して公開する。

## What Changes

- Zenn 記事の執筆(背景 → 実験設計 → 実装 → 結果 → 考察 → 結論)
- 図表の作成(達成率比較、失敗モード分布)
- GitHub リポジトリの整備(README、コード整理、結果データ)

## Capabilities

### New Capabilities

- `article`: Zenn 記事(Markdown + 図表)
- `public-repository`: 公開可能な GitHub リポジトリ(コード + 結果 + README)

### Modified Capabilities

(なし)

- 執筆期間: 4-5 日
